ADK : framework com plataforma de engenharia de software para agentes. Tem para python, js

Criando ambiente python: python -m venv .venv
Instalando no ambiente: pip install google-adk

Criando agente ADK com linha de comando:adk createt my_agent
1 , 1 

Cola o link da chave do API: https://aistudio.google.com/api-keys?project=gen-lang-client-0312672053

adk run my_agent
ou
adk web
